# Smart Study Notes

A clean, minimal study notes manager built using React.  
You can add, search, and manage study notes in one place, with a simple dashboard for focused learning.

## Features

- Add new notes with a title and content  
- See all your notes in a pastel card layout  
- Mark notes as **done** or **pending**  
- Delete notes you no longer need  
- Search notes by title or content  
- Small dashboard area for:
  - **Smart Study Mode** (timer placeholder)
  - **This Week** study time (placeholder for future stats)

## Tech Stack

- **React** (Create React App)
- **JavaScript (ES6+)**
- **CSS** (custom, Notion-like soft UI)
