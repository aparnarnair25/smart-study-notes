# Smart Study Notes

Smart Study Notes is a React-based web application designed to help students
organize their study notes and track learning tasks in a clean, distraction-free
environment.

## Problem Statement
Students often struggle to manage multiple notes and study tasks using
different tools. This application provides a single platform to create, track,
and organize study notes efficiently.

## Features
- Add new notes with a title and content
- View all notes in a soft pastel card layout
- Mark notes as done or pending
- Delete notes that are no longer needed
- Search notes by title or content
- Mini dashboard section:
  - Smart Study Mode (timer placeholder)
  - This Week Study Time (future analytics placeholder)

## Tech Stack
- React (Create React App)
- JavaScript (ES6+)
- CSS (custom Notion-like UI)

## System Flow
User → React UI → Application Logic → State Management → UI Update

## How to Run Locally
1. Clone the repository:
   git clone https://github.com/aparnarnair25/smart-study-notes.git

2. Install dependencies:
   npm install

3. Start the development server:
   npm start
